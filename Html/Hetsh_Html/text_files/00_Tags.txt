Complete List of Semantic HTML Elements
1. Structural Elements
Element	Description
<html>	Root element of an HTML document
<head>	Contains metadata, scripts, styles, etc.
<body>	Contains the visible content of the document
<header>	Introductory content or navigational links
<nav>	Section with navigation links
<main>	Main content unique to the document
<section>	Thematic grouping of content
<article>	Independent, self-contained content (blog post, news)
<aside>	Content aside from main content (sidebar, callout)
<footer>	Footer for a section or page
<address>	Contact information
________________________________________
2. Text Content Elements
Element	Description
<h1> - <h6>	Headings of different levels
<p>	Paragraph
<hr>	Thematic break (horizontal rule)
<pre>	Preformatted text
<blockquote>	Section quoted from another source
<cite>	Citation of a creative work
<code>	Inline code snippet
<samp>	Sample output from a program
<kbd>	Keyboard input
<var>	Variable or placeholder text
<mark>	Highlighted text
<del>	Deleted text
<ins>	Inserted text
________________________________________
3. Inline Text Semantics
Element	Description
<a>	Hyperlink
<strong>	Strong importance (usually bold)
<em>	Emphasis (usually italic)
<b>	Bold text (without extra importance)
<i>	Italic text (without extra emphasis)
<u>	Underlined text
<small>	Smaller text (fine print)
<sup>	Superscript
<sub>	Subscript
________________________________________
4. Embedded Content
Element	Description
<img>	Image
<video>	Video player
<audio>	Audio player
<source>	Media resource for <video> or <audio>
<track>	Text tracks (captions, subtitles) for media
<iframe>	Inline frame (embedding another webpage)
<embed>	Embedded content like plugins
<object>	Generic embedded object
<picture>	Responsive images
________________________________________
5. Table Elements
Element	Description
<table>	Table container
<caption>	Table caption
<thead>	Table header group
<tbody>	Table body group
<tfoot>	Table footer group
<tr>	Table row
<th>	Header cell
<td>	Data cell
________________________________________
6. Form Elements
Element	Description
<form>	Form container
<label>	Label for form input
<input>	Various input types (text, checkbox, etc.)
<textarea>	Multi-line text input
<button>	Clickable button
<select>	Drop-down list
<option>	Option in a drop-down
<fieldset>	Group related inputs
<legend>	Caption for a fieldset
<datalist>	Predefined list of options
________________________________________
7. Other Useful Semantic Elements
Element	Description
<details>	Disclosure widget, can show/hide content
<summary>	Summary or label for <details>
<dialog>	Dialog box or modal window
<meter>	Scalar measurement within a known range
<progress>	Progress of a task
<time>	Date or time value
________________________________________
