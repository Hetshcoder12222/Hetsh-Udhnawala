Link Tag

We use link tag to link external resources like CSS/JavaScript files.


rel means relationship, and href means the path to the file.


rel is used to define the relationship between the current document and the linked resource and for its type like js is script, css is stylesheet etc.


We can use the ./ to refer to the current directory, or ../ to refer to the parent directory.


Meta Tags

UTF tag

The meta tag is used to provide metadata about the HTML document.


The charset attribute specifies the character encoding for the HTML document.


UTF-8 dont support emojis, while UTF-16 supports emojis.


Keywords tag

The meta keyword tag in HTML is used to provide information about a web page to search engines, helping them understand the page's content and relevance.


Description tag

The meta description tag in HTML is used to provide a brief summary of a web page's content, which can be displayed in search engine results.


Http-equiv tag

The http-equiv attribute in the meta tag is used to provide HTTP headers in the HTML document.


If the http-equiv attribute is set, the element is a pragma directive, providing information equivalent to what can be given by a similarly-named HTTP header.

Viewport tag

The viewport meta tag is used to control the layout and scaling of a web page on mobile devices.


The device width sets the width of the page to follow the screen-width of the device


The initial scale sets the zoom level
